# Доступ с телефона по HTTPS (камера)

Камера на телефоне работает только по **HTTPS**. Ниже — как запустить приложение с HTTPS и зайти с телефона.

---

## Шаг 1: Сгенерировать сертификаты

```bash
npm run generate-certs
```

Создаются файлы в `server/certs/` (key.pem, cert.pem). Они нужны для HTTPS.

---

## Шаг 2: Собрать клиент и запустить сервер по HTTPS

```bash
npm run prod:https
```

В консоли будет сообщение: `HTTPS Server on port 3000`.

---

## Шаг 3: Узнать IP-адрес компьютера

**macOS:**
```bash
ipconfig getifaddr en0
```
или
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

**Windows:**
```cmd
ipconfig
```
Смотрите адрес вроде `192.168.1.100` для Wi‑Fi.

---

## Шаг 4: Открыть приложение на телефоне

1. Подключите телефон к той же Wi‑Fi, что и компьютер.
2. Откройте браузер на телефоне.
3. Введите адрес (подставьте свой IP):
   ```
   https://192.168.1.100:3000
   ```
4. Браузер покажет предупреждение о самоподписанном сертификате.
5. Нажмите **«Дополнительно»** → **«Перейти на сайт»** (или аналогичный вариант).
6. Откройте приложение, авторизуйтесь и используйте камеру для фото.

---

## Примечания

- Телефон и компьютер должны быть в одной сети (тот же Wi‑Fi).
- Порт 3000 не должен быть закрыт фаерволом.
- Сертификат самоподписанный — предупреждение в браузере нормально.
